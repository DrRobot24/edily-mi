# Edily Milano - Luxury Construction & Real Estate

![Edily Milano](/public/images/Logo.png)

**Edily Milano** is a premium, single-page web application designed for a luxury construction and real estate firm based in Milan. The interface is engineered to reflect high-end architectural aesthetics, combining minimalist "dark luxury" styling with cinematic scroll animations.

## 🌟 Key Features

- **Cinematic Presentation:** Employs advanced smooth scrolling (via Lenis) and scroll-triggered animations (via GSAP) to deliver a high-budget feel.
- **Dynamic Navigation:** Features a state-of-the-art transparent "headbar" that dynamically transitions to a stark white header mechanism with inverted logo and text constraints upon scrolling.
- **Bespoke Interactions:** Natively includes premium parallax image effects and custom layout flows tailored specifically for high-net-worth real estate showcases.
- **Fully Responsive:** Seamlessly degrades layout and interactive elements for mobile and tablet viewers, employing a custom modal navigation.

## 🛠️ Technology Stack

- **Framework:** Vite
- **Structure:** Vanilla HTML5, CSS3, ES6 JavaScript
- **Animations:** GSAP (GreenSock Animation Platform) + ScrollTrigger
- **Scrolling Dynamics:** Lenis
- **Package Manager:** npm

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/DrRobot24/edily-mi.git
   cd edily-mi
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the Development Server:**
   ```bash
   npm run dev
   ```

4. **Build for Production:**
   ```bash
   npm run build
   ```
   The optimized production files will be output to the `dist/` directory.
